## 科技lion大佬的k命令

### 更新系统
``` bash
apt-get update
```

### 安装必要工具
``` bash
apt-get install -y xz-utils openssl gawk file
```

### 科技lion脚本工具箱
``` bash
curl -sS -O https://kejilion.pro/kejilion.sh && chmod +x kejilion.sh && ./kejilion.sh
```

``` bash 
bash <(curl -sL kejilion.sh)
```

k命令主菜单
